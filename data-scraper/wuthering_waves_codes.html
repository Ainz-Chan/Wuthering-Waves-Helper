Wuthering Waves Codes

All Active Promo Codes

WUTHERINGGIFT is currently the only active Wuthering Waves redeem code. The code will get you a nice bit of Astrite to use for the Gacha system, along with other valuable in-game materials.

Redemption Codes Rewards
WUTHERINGGIFT ● Astrite x50
● Premium Resonance Potion x2
● Medium Revival Inhaler x2
● Medium Energy Bag x2
● Shell Credit x15000

How to Redeem Codes in Wuthering Waves

Use the Redeem Function Found in the Game Settings

  1. Open the settings and go to "Other Settings"
  2. Use the "Redemption Code" feature
  3. Enter your redeem code and confirm
  4. Open your mailbox to claim the rewards

Open your in-game settings and click on the Other Settings tab at the bottom-left part of the menu. From there, you can click on the "Redeem" button to use the in-game Redemption Code feature.

Type in your active Redeem Code and confirm. A pop-up will appear to let you know that the code was claimed successfully. The rewards will be in your in-game mailbox.

Claim the Code Rewards Immediately


The mail containing the redeem code rewards will automatically be deleted after two weeks, so you should claim the goods as soon as you can!

Can You Redeem Wuthering Waves Codes on a Website?

No Way to Redeem Codes on a Website

The official Wuthering Waves website currently has no feature for redeeming codes. The in-game method is the only way to redeem codes and rewards as of now.
